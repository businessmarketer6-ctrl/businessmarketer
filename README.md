# Business Marketer — Digital Marketing Agency in India

A single-page static website (GitHub Pages ready).

## Files
- `index.html` — page markup
- `style.css` — all styling
- `script.js` — scroll reveal, mobile menu, FAQ accordion
- `images/` — SVG graphics (hero, channel illustrations, favicon)

## Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files (keep the `images/` folder intact).
3. Repo → **Settings → Pages** → Source: `Deploy from a branch` → Branch: `main` → `/ (root)` → Save.
4. Your site goes live at `https://<username>.github.io/<repo>/`.

## Custom domain (optional)
Add a file named `CNAME` containing your domain (e.g. `businessmarketer.in`).

All links point to https://businessmarketer.in/
